## ROCK PAPER SCISSORS
An HTML5 and CSS project using the "Holy Grail" layout
using flexbox and box model concepts.

## Skills Demonstrated

Javascript:
- Appropriate variable naming and usage conventions(const, let, upperCase for const);
- Appropriate usage of function declaration(hoisting) and arrow functions(helpers)
- Incrementing variables++
- Using stricy conditionals ===, ||, and &&
- Using confirm to ask an alert and use window.close if cancel in playing
- Using an array in choices and used Math.random() to determine computer choice

DOM:
- Creating HTML elements via createElement
- Creating DOM variables via querySelectors
- Usage of textContent and appendChild to change displays
- Using span to allow changing of scores via textContent and appendChild

Events:
- Using event listeners and callbacks to not repeat logic of the buttons via delegation

Git:
- Branching, checking another branch, merging, and deleting the branch.
